# Retail-Sales-Performance-Dashboard

## 📌 Project Overview

Developed an interactive Retail Sales Dashboard using Excel, MySQL, and Power BI. Cleaned and transformed raw data in Excel, analyzed it using SQL queries, and built interactive dashboards with DAX measures, KPIs, and visualizations to provide insights into sales performance, revenue trends, and product categories.

## 🎯 Objectives

* Clean and transform raw retail sales data for analysis.
* Analyze sales data using SQL queries.
* Build an interactive Power BI dashboard with DAX measures and KPIs.
* Visualize sales trends, revenue, and product performance.
* Enable data-driven business decision-making through interactive reports.

## 🛠️ Tools & Technologies
* Microsoft Excel – Data cleaning and preprocessing
* MySQL – Data storage and SQL querying
* Power BI – Dashboard creation and data visualization
* DAX (Data Analysis Expressions) – KPIs and calculated measures

## 📈 Dashboard Features
* KPI Cards for Total Revenue, Total Quantity, Total Transactions, Highest Sale, and Average Order Value
* Monthly Revenue Trend Analysis
* Revenue Analysis by Product Category
* Revenue Distribution by Gender
* Quantity Sold by Product Category
* Interactive Filters and Slicers for Product Category and Gender
* Dynamic Dashboard with DAX Measures and Business KPIs

## 📊 Key Insights
* Electronics generated the highest revenue among all product categories.
* Revenue trends highlight monthly sales performance and seasonal variations.
* Sales distribution by gender provides insights into customer purchasing behavior.
* KPI metrics help track total revenue, transactions, quantity sold, and average order value.
* Interactive filters enable detailed analysis by product category and gender, supporting data-driven business decisions.

## 🚀 How to Use
* Download or clone this repository.
* Open the Retail Sales Dashboard (.pbix) file in Power BI Desktop.
* Refresh the data if required.
* Use the Product Category and Gender slicers to explore sales insights.
* Analyze KPIs, revenue trends, and product performance through the interactive dashboard.
